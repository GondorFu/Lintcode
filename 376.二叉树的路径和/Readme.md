# GondorFu
---
## 描述
给定一个二叉树，找出所有路径中各节点相加总和等于给定 目标值 的路径。

一个有效的路径，指的是从根节点到叶节点的路径。

## 样例
```
给定一个二叉树，和 目标值 = 5:

     1
    / \
   2   4
  / \
 2   3
返回：

[
  [1, 2, 2],
  [1, 4]
]
```

## 标签 
二叉树 二叉树遍历

## 解题思路
- 思路：
    
    使用递归计算各子树和是否满足对应数值要求，主要注意链表和数之间的判断。
    
- 具体操作：

    具体方法见程序。
