# GondorFu
---
## 描述：
给定n个不同的正整数，整数k（k < = n）以及一个目标数字。　　　　

在这n个数里面找出K个数，使得这K个数的和等于目标数字，求问有多少种方案？

## 样例
```
给出[1,2,3,4]，k=2， target=5，[1,4] and [2,3]是2个符合要求的方案
```

## 标签 
动态规划

## 解题思路
- 思想：
    这是一道典型的动态规划问题：通过把原问题分解为相对简单的子问题的方式求解复杂问题的方法。

    动态规划方法的关键是找到状态转移方程，即当前问题能够用一个或几个规模更小的问题替代。比如本题中的状态转移方程就是：

    [n,k,t] = [n-1,k,t] + [n-1,k-1,t-A[n-1]]
    其中[n,k,t]表示数组前n个数中选取k个数等于t的方法数。

    这样关于公式的正确性可以自己理解。
    得到具体状态转移方程之后，就能很方便的实现了。
    

- 具体操作：
    可以使用三维数组进行存储，不过发现状态转移方程只涉及上一时刻（n-1）的状态，所以只需要用一个二维数组保存[k,t]即可，为一个(k+1)*(t+1)的dp数组。

    初始化：由于在最开始时只选取数组中前0个元素，即没有元素被选，所以除dp[0][0]=1外其他元素都为零。

    之后每一步增加一个数组的元素，然后遍历整个dp数据，根据状态转移方程不断更新即可。

    最后的结果就在dp[k][target]中。
