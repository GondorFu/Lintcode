# GondorFu
---
## 描述
给一个由 1 - n 的整数随机组成的一个字符串序列，其中丢失了一个整数，请找到它。

- 注意事项
n <= 30

## 样例
```
给出 n = 20, str = 19201234567891011121314151618

丢失的数是 17 ，返回这个数。
```

## 标签 
回溯法 深度优先搜索

## 解题思路
- 思路：
    
    由于每个数只有两种可能，即单独成数或者与后一个数结合。因此使用深度优先搜索，如果可以构成两位数，优先组合成两位数，并用堆栈存储两位数位置。如果出现矛盾，则跳转到最近的两位数处尝试一位数。
    
- 具体操作：

    在操作过程中需要注意两点：
	1. 在跳转回之前的两位数位置时，需要将之间所有的一位数和跳转位置的两位数，取消标记，因为它们已没有被发现。
	2. 最后结果可能是一个两位数，但是却将两个一位数合并为了这个丢失的两位数，造成有两个个位数不存在。比如丢失的是27，但个位数2和7刚好能够组合成27，但这样丢失了两个数2和7。而且注意：这个二位数的另一种组合一定不存在，因为那样将无法分辨。比如丢失的12由1和2组成，这时将无法判断丢失的到底是12还是21.
