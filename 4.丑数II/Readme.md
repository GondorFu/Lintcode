# GondorFu
---
## 描述
设计一个算法，找出只含素因子2，3，5 的第 n 大的数。

符合条件的数如：1, 2, 3, 4, 5, 6, 8, 9, 10, 12...

- 注意事项

    我们可以认为1也是一个丑数

## 样例
```
如果n = 9， 返回 10
```

## 挑战 
要求时间复杂度为O(nlogn)或者O(n)

## 标签 
优先队列

## 解题思路
- 思路：
    
    维持一个最新添加的丑数*2，*3，*5的三个数组
    
- 具体操作：

    m2，m3, m5 初始化为2, 3, 5，对应指针为0,0,0.之后重复的选择两个数组对应指针中数字最小的数加入丑数队列。为防止三个数组中元素重复，从m3中取数时，只将对应元素*3，*5加入m3,m5，m5时类似。之后不断重复，直到取到第n个数为止。
