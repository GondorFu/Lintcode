# GondorFu
---
## 描述
实现一个带有取最小值min方法的栈，min方法将返回当前栈中的最小值。

你实现的栈将支持push，pop 和 min 操作，所有操作要求都在O(1)时间内完成。

- 注意事项

    如果堆栈中没有数字则不能进行min方法的调用

## 样例
```
如下操作：push(1)，pop()，push(2)，push(3)，min()， push(1)，min() 返回 1，2，1
```

## 标签 
栈 谷歌 优步 Zenefits

## 解题思路
- 思路：
    
    使用两个栈分别存储原数据s1和现有最小数据sm。取最小数据直接取最小数据栈的栈顶元素即可。同理可以扩展到最大元素。
    
- 具体操作：

    当压入元素时，如果为空，直接将原数据压入两个栈。否则，将原数据压入s1，并比较原数据和sm的栈顶元素，如果小于则压入原数据，如果大于等于则将栈顶元素复制压入sm。弹出直接从两堆栈弹出数据即可。
