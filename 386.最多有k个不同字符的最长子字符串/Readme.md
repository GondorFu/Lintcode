# GondorFu
---
## 描述
给定一个字符串，找到最多有k个不同字符的最长子字符串。

## 样例
```
例如，给定 s = "eceba" , k = 3,

T 是 "eceb"，长度为 4.
```

## 挑战 
O(n), n 是所给字符串的长度

## 标签 
两根指针 哈希表 字符串处理

## 解题思路
- 思路：
    首先使用两个指针标记选取的子字符串，其中用哈希存储出现过的字符和出现的次数。如果哈希表的大小大于k（k+1），就需要删除开头的元素，i=i+1，即哈希表中对应字符出现的次数减一。如果出现次数变为零，即可删除这个字符。此时，i，j选择的子字符串将满足要求。
    
    这样遍历一遍字符串不断更新最长子字符串的大小就能够得到最后的结构。

    具体操作见思路。
