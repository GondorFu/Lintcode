# GondorFu
---
## 描述

每个学生有两个属性 id 和 scores。找到每个学生最高的5个分数的平均值。

## 样例
```
给出 results = [[1,91],[1,92],[2,93],[2,99],[2,98],[2,97],[1,60],[1,58],[2,100],[1,61]]

返回：
1: 72.40
2: 97.40
```

## 标签 
堆 亚马逊

## 解题思路
- 思路：
    
    不断取最优数使用堆。然后使用一个数组存放每个同学的堆成绩。
    
- 具体操作：

    见程序。不过需要注意堆的操作中一定要考虑元素比较值相等的情况。
