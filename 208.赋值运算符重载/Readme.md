# GondorFu
---
## 描述
实现赋值运算符重载函数，确保：

新的数据可准确地被复制
旧的数据可准确地删除/释放
可进行 A = B = C 赋值

## 说明
本题只适用于C++，因为 Java 和 Python 没有对赋值运算符的重载机制。

## 样例
```
如果进行 A = B 赋值，则 A 中的数据被删除，取而代之的是 B 中的数据。

如果进行 A = B = C 赋值，则 A 和 B 都复制了 C 中的数据。
```

## 挑战 
充分考虑安全问题，并注意释放旧数据。

## 标签 
面向对象设计

- 思路：
    
    注意释放原有空间，并新建新空间。
