# GondorFu
---
## 描述
给你一棵二叉树，找二叉树中的一棵子树，他的所有节点之和最大。

返回这棵子树的根节点。

- 注意事项
    LintCode 会把你返回的节点作为最优子树来打印。

    数据保证有且仅有唯一的解。

## 样例
```
给出如下二叉树

     1
   /   \
 -5     2
 / \   /  \
0   3 -4  -5 
返回值为 3 的节点。
```

## 标签 
二叉树 亚马逊 深度优先搜索

## 解题思路
- 思路：
    
    通过递归遍历树并返回子树的和值，最大和值节点和最大和值。
    
- 具体操作：

    通过后序遍历树，并根据遍历情况分情况讨论。
    1. 如果左右返回为空，为叶子节点，将该节点返回
    2. 如果左为空，判断和值是否大于子树最大和值，如果大于返回根节点，否则返回子树最大和值节点。右为空类似。
    3. 如果左右都不为空，同样根据第二点返回最大和值节点。
