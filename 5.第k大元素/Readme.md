# GondorFu
---
## 描述
在数组中找到第k大的元素

- 注意事项

    你可以交换数组中的元素的位置

## 样例
```
给出数组 [9,3,2,4,8]，第三大的元素是 4

给出数组 [1,2,3,4,5]，第一大的元素是 5，第二大的元素是 4，第三大的元素是 3，以此类推
```

## 挑战 
要求时间复杂度为O(n)，空间复杂度为O(1)

## 标签 
排序 快速排序

## 解题思路
- 思路：
    
    借助快速排序中数组划分的思想，随机选择一个元素，并把数组分为小于这个元素和大于等于这个元素的两个子数组。这样就可以知道这个元素是数组的第几大元素了。之后再用递归的方法找出之前数组的第k大和之后数组的第k-n大就可以得到结果了。
