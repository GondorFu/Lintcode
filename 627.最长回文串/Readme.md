# GondorFu
---
## 描述
给出一个包含大小写字母的字符串。求出由这些字母构成的最长的回文串的长度是多少。

数据是大小写敏感的，也就是说，"Aa" 并不会被认为是一个回文串。

- 注意事项
    假设字符串的长度不会超过 1010。

## 样例
```
给出 s = "abccccdd" 返回 7

一种可以构建出来的最长回文串方案是 "dccaccd"。
```

## 标签 
哈希表 亚马逊

## 解题思路
- 思路：
    
    如果字符出现的次数为偶数，则可以完全添加。如果为奇数，则只能有一个唯一的奇数字符能够被放在字符串中间而全部被添加，而其他的只能删除一个字符。
    
- 具体操作：

    使用哈希记录每个字符出现的次数。之后统计能被使用的数目。如果出现次数为偶数，那么能够被全部添加。如果出现的次数为奇数，则需要对是否之前出现过奇数情况进行判断，如果未出现则全部添加，如果出现过，那么只能删除一个元素之后再全部添加。
