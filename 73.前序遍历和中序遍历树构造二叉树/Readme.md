# GondorFu
---
## 描述
根据前序遍历和中序遍历树构造二叉树.

- 注意事项
你可以假设树中不存在相同数值的节点

## 样例
```
给出中序遍历：[1,2,3]和前序遍历：[2,1,3]. 返回如下的树:

  2
 / \
1   3
```

## 标签 
二叉树

## 解题思路
- 思路：

    对于树第一个想到的就应该是递归，而根据给出的条件可以很方便的得到对应的左右子树。
    
- 具体操作：

    每次迭代去preorder的第一个元素为根节点，而对应的左子树对应preorder中的[1:index]和inorder中的[0:index-1]，而右子树对应preorder中的[index+1:end]和inorder中的[index+1:end]。这样在丢用buildTree函数分别处理左右子树就能得到最后的结果了。
