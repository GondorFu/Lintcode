# GondorFu
---
## 描述
给定一个字符串和一个偏移量，根据偏移量旋转字符串(从左向右旋转)

## 样例
```
对于字符串 "abcdefg".

offset=0 => "abcdefg"
offset=1 => "gabcdef"
offset=2 => "fgabcde"
offset=3 => "efgabcd"
```

## 挑战 
在数组上原地旋转，使用O(1)的额外空间

## 标签 
字符串处理

## 解题思路
- 思路：
    
    分别对前后两个字符串进行翻转，之后再对整个字符串进行翻转即可。
    
- 具体操作：

    见程序。
