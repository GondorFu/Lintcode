# GondorFu
---
## 描述
给定两个矩形，判断这两个矩形是否有重叠。

- 注意事项
    l1代表第一个矩形的左上角
    
    r1代表第一个矩形的右下角
    
    l2代表第二个矩形的左上角
    
    r2代表第二个矩形的右下角

    保证：l1 != r2 并且 l2 != r2

## 样例
```
给定 l1 = [0, 8], r1 = [8, 0], l2 = [6, 6], r2 = [10, 0], 返回 true

给定 l1 = [0, 8], r1 = [8, 0], l2 = [9, 6], r2 = [10, 0], 返回 false
```

## 标签 
数学 亚马逊

## 解题思路
- 思路：
    
    只需要画图逻辑分析一下，只要一个矩阵位于另一个矩阵的完全上下左右侧，那么它们分离，否则它们重合。
    
- 具体操作：

    见程序。
