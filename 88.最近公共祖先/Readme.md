# GondorFu
---
## 描述：
给定一棵二叉树，找到两个节点的最近公共父节点(LCA)。

最近公共祖先是两个节点的公共的祖先节点且具有最大深度。

- 注意事项

    假设给出的两个节点都在树中存在

## 样例
```
对于下面这棵二叉树

  4
 / \
3   7
   / \
  5   6
LCA(3, 5) = 4

LCA(5, 6) = 7

LCA(6, 7) = 7
```

## 标签 
领英 二叉树 脸书

## 解题思路
- 思想：
    
    分析问题我们能够得出这些的结论，在两数一定存在在树中的前提下：
    1. 对于某一个根节点，如果两数分别位于这个节点左右子树，那么该根节点就是最近公共祖先，直接返回它。
    2. 如果某个节点就是要找的某一个数，一种情况是另一个数在该节点的子树里，那么该节点就是最近公共祖先；而如果另一个数在其他分支中，那么按照结论1，必然存在一个根节点，两个数分别位于它的左右子树中。所以当遍历到某个节点就是所要找的数时，不需要继续对其子树进行搜索，只要直接返回该节点即可。
    3. 如果是由结论1返回的根节点，那么它之上将不会存在某个节点，两个数分别位于它的左右两个子树。返回的最近公共祖先将被维持。
    
- 具体操作：
    
    依据思路中总结的结论，我们能够进行使用递归求解问题。对于某一层递归，首先需要判断根节点是否为要找元素，如果是，根据结论2可以直接返回该根节点。如果不是，那么根据结论1，分别递归搜索它的左右两个子树。
    
    搜索结束后，如果左右两数都有返回结果，根据结论1说明该根节点就是最近公共祖先。如果只有一个子树有返回结果，那么两数都应该在该子树中，而返回值就是最近公共祖先。如果都没有返回结果，说明两数都不在该子树中，为了区分，返回Null。
