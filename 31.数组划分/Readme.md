# GondorFu
---
## 描述：
给出一个整数数组 nums 和一个整数 k。划分数组（即移动数组 nums 中的元素），使得：

所有小于k的元素移到左边
所有大于等于k的元素移到右边
返回数组划分的位置，即数组中第一个位置 i，满足 nums[i] 大于等于 k。

- 注意事项

    你应该真正的划分数组 nums，而不仅仅只是计算比 k 小的整数数，如果数组 nums 中的所有元素都比 k 小，则返回 nums.length。

## 样例
```
给出数组 nums = [3,2,2,1] 和 k = 2，返回 1.
```

## 挑战 
使用 O(n) 的时间复杂度在数组上进行划分。

## 标签 
排序 两根指针 数组

## 解题思路
- 思想：
    
    数组划分是快速排序的一个操作步骤。主要借助两个指针分别标记<k的数组部分和>=k的数组部分。
    
- 具体操作：

    首先定义两个指针i,j分别指向数组的首尾。然后如果i位置的数<k就i++，如果j位置的数>=k就j--。然后交换i,j两位置的数值。之后重复进行指导i>=j位置。
    
    这里需要注意一下边界条件，即所有元素都<k的情况。不过此时只要判断出如果i==len(n)-1时，就表示出现所述特殊情况，在最后进行判断就能解决了。
