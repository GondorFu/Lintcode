# GondorFu
---
## 描述：
假定用一个链表表示两个数，其中每个节点仅包含一个数字。假设这两个数的数字顺序排列，请设计一种方法将两个数相加，并将其结果表现为链表的形式。

## 样例
```
给出 6->1->7 + 2->9->5。即，617 + 295。

返回 9->1->2。即，912 。
```

## 标签 
链表 高精度

## 解题思路
- 思想：
    
    可以使用直接计算，判断各位和是否大于或等于9，然后不断判断。但这样需要提前统计链表长度，才能对齐处理。所以我直接使用35的翻转链表和167的链表求和程序，通过翻转链表-求和-再翻转链表得到了结果。
    
- 具体操作：

    参考35.翻转链表和167.链表求和程序说明。
