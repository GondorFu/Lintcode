# GondorFu
---
## 描述
给定一个含不同整数的集合，返回其所有的子集

- 注意事项

    子集中的元素排列必须是非降序的，解集必须不包含重复的子集

## 样例
```
如果 S = [1,2,3]，有如下的解：

[
  [3],
  [1],
  [2],
  [1,2,3],
  [1,3],
  [2,3],
  [1,2],
  []
]
```

## 挑战
你可以同时用递归与非递归的方式解决么？

## 标签 
递归 脸书 优步

## 解题思路
- 思路：
    
    先排序然后从后前递归子数组，再对于递归结果选择添加或不添加最后一个元素两种情况得到全部子集。
    
- 具体操作：

    在计算过程中如果将rlt.append(v + [S[-1]])替换为v.append(S[-1]) rlt.append(v),由于rlt中的srlt部分，v和srlt其中共用一块空间，rlt中的相应元素也会发生改变。
