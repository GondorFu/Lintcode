# GondorFu
---
## 描述
有 n 个硬币排成一条线。两个参赛者轮流从右边依次拿走 1 或 2 个硬币，直到没有硬币为止。拿到最后一枚硬币的人获胜。

请判定 第一个玩家 是输还是赢？

## 样例
```
n = 1, 返回 true.

n = 2, 返回 true.

n = 3, 返回 false.

n = 4, 返回 true.

n = 5, 返回 true.
```

## 挑战 
O(1) 时间复杂度且O(1) 存储。

## 标签 
动态规划 贪心 数组 博弈论

- 思路：
    
    其实状态转移方程A[n]= !(A[n-1] and A[n-2]) A[0] = False
    
    看到挑战，直接开始找规律。以3为倍数，输。
