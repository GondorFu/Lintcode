# GondorFu
---
## 描述
有两个不同大小的二进制树： T1 有上百万的节点； T2 有好几百的节点。请设计一种算法，判定 T2 是否为 T1的子树。

- 注意事项
    
    若 T1 中存在从节点 n 开始的子树与 T2 相同，我们称 T2 是 T1 的子树。也就是说，如果在 T1 节点 n 处将树砍断，砍断的部分将与 T2 完全相同。

## 样例
```
       1                3
      / \              / 
T1 = 2   3      T2 =  4
        /
       4
下面的例子中 T2 不是 T1 的子树：

       1               3
      / \               \
T1 = 2   3       T2 =    4
        /
       4
```

## 标签 
递归 二叉树

## 解题思路
- 思路：
    
    直接使用递归对T1及其子树判断是否与T2相同。
    
- 具体操作：

    在具体操作时会发现，在已确定子树部分与T2相同后，如果出现不同，需要重新对子树判断是否完全与T2相同。而此时T2已经为T2的子树，因此需要将递归程序另写一个函数，再在原函数中存储T2并调用T2即可。
